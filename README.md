# BlogAggreGator Project for Boot.dev

I enjoyed making this. It was actually one of my first ever projects using a whole stack and it really opened my eyes to a few mistakes I used to always make.

## How to use:

- Enter the commands:  npm run start [command] [args]

Replacing [command] and [args] with your own input.

### Commands:

- login | [user] (replacing [user] with a username)
- register | [user] (replacing [user] with a username)
- reset (Reset the database (gator))
- users (List all users in the database (gator))
- agg (Scan all feeds within a certain timeframe and print them)
- addfeed | [url] (Add an RSSFeed by replacing [url] with the feeds URL)

- feeds (Scan all feeds and print them)
- follow | [url] (Replacing URL with a feed URL)
- unfollow | [url] (Replacing URL with a feed URL)
- browse | [limit] (See the most recent followed feeds up to a certain limit)
